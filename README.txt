Tree Valuation
==============

** This tool is still Work In Progress **

Automates form filling when having a bunch of trees  to  valuate.   The  program
expects trees to be in a Excell file.  It parses the file and valuates each tree
via endpoint located in [2].  It than writes the returned data to a Excell file.
